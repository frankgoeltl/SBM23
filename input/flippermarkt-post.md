**Titel:** Silverball Mania — eigene Regeln im Browser testen

---

Hallo zusammen,

für meinen 1980er **Silverball Mania** habe ich auf Basis von Dick Hamills [RPU](https://www.pinballrefresh.com/retro-pin-upgrade-rpu) einen eigenen Regelsatz gebaut und dazu einen **Browser-Simulator**, in dem sich drei Regelsätze direkt vergleichen lassen:

- **1980** — werksseitiges Original
- **SBM23** — erweiterter Regelsatz von Dick Hamill
- **FRG** — mein eigener Twist (Horseshoe spottet nur MANIA, Top Lanes advancen Bonus X, aufgemotzte Bumper, Spinner-Jackpot am Horseshoe einsammelbar)

Kein Physik-Simulator, sondern eine Regel-Sandbox: Schalter anklicken, Scoring live beobachten.

**Link:** https://hagen.vercel.app *(URL nach Deployment anpassen)*

**Transparenz:** Ich habe keine Zeile Code selbst geschrieben — der komplette Simulator ist mit **Claude Code** im Dialog entstanden. Details dazu auf der About-Seite.

Feedback zum FRG-Regelsatz gern willkommen — vor allem ob sich der Spinner-Jackpot richtig anfühlt.

Grüße, Frank
